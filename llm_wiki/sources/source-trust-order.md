# Source Trust Order

Use this order when selecting citations:

1. Official federal/state/county/city sources.
2. Official application portals and official no-key APIs.
3. Official local service guides.
4. 211, food-bank, and legal-aid nonprofit handoff pages.
5. Google Maps/Places/Routes only for contact/location enrichment, not benefits
   logic.

Final packets must cite approved source IDs and approved domains only.
