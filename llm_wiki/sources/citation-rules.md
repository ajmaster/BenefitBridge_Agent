# Citation Rules

Every user-facing benefit path, local handoff, and packet export must cite
approved source metadata.

Required citation fields:

- `source_id`
- `title`
- `url`
- `owner_type`
- `last_checked`
- `freshness_state`

Rules:

- Prefer official state, county, city, or federal sources over partner sources.
- Use partner or nonprofit sources for plain-language support only when the
  packet also cites official program sources.
- Never invent a URL or cite an unapproved domain.
- Do not cite visual references, screenshots, or generated design assets for
  policy facts.
- Do not quote long source text. Summarize and point to the citation.
- If a source is stale and critical, block export or mark the path as needing
  source refresh.
