# License and Source Notes

This pack stores source metadata, URLs, short summaries, and limited safe claims. It does not redistribute large page copies.

## Guidance

- Prefer linking to official pages instead of copying content.
- Keep snippets short and source-labeled.
- Cache only what the agent needs for grounding/evals.
- Respect site terms for 211, food bank, SF Service Guide, and nonprofit directories.
- Use Google Places/Routes for enrichment only; do not replace official benefit or shelter sources.

## Open standards

Open Referral HSDS is used as an internal schema inspiration for human-service resources. HSDS is not itself a data source.
